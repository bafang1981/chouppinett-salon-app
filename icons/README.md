# PWA Icons for StrandPro

This directory contains the PWA icons for StrandPro Salon Management application.

## Required Icons
Please add the following icon files based on the StrandPro logo:

- `icon-72x72.png` - 72x72 pixels
- `icon-96x96.png` - 96x96 pixels  
- `icon-128x128.png` - 128x128 pixels
- `icon-144x144.png` - 144x144 pixels
- `icon-152x152.png` - 152x152 pixels
- `icon-192x192.png` - 192x192 pixels
- `icon-384x384.png` - 384x384 pixels
- `icon-512x512.png` - 512x512 pixels

## Icon Design Guidelines
- Use the StrandPro logo with golden/orange and silver colors
- Background should be transparent for maskable icons
- Ensure good contrast and readability at small sizes
- The golden color (#C9922A) should be prominent
- Consider adding a subtle background for better visibility

## Temporary Placeholders
For development, you can use SVG placeholders or simple colored squares with the "SP" text until the actual icons are created.
